My name is Matthew LeBar, and I am a Master's Student in Computer Science at the University of Chicago, where I am focusing on Machine Learning and Data Analytics, as well as training my computer science fundamentals. Odin-Book

Webpage Codebase

Built with Node, Express, and MongoDB. This was a complicated project with a lot of moving parts - using Faker.js to populate with fake data, using Passport.js and Bcrypt.js to authenticate login information. I'm proud of how it all came together!

Spirit-GPT

Codebase

A machine learning project, replicating the style of 19th century german philosopher Hegel's writing by following a tutorial by Andrej Karpathy (see README for the link to the tutorial). After a few runs on google colab I got a nice, if not entirely intelligible result.

Zoo Shop

Webpage Codebase

Memory Card

Webpage Codebase

CV Project

Webpage Codebase

This was my first project in React, and a complex one behind the scenes. My focus was on making sure everything functioned exactly as it should - it was difficult to develop code flexible enough to accomodate all the possible inputs (since you could have any number of degrees or jobs), while rigid enough to provide them the right formatting.
